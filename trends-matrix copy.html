QEEPP Visual Models

QEEPP Trends

A visual model used to track repeated QEEPP assessments over time across Quality, Effectiveness, Efficiency, Performance, and Productivity.

Each matrix represents one assessment period. Each column fills vertically from level 1 to level 5, accordingly to the scorecard for the assesment, making movement across periods easy to compare.

Enter four repeated assessment snapshots. Each row controls an associated matrix in the trend chart.
